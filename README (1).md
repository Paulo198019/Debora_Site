# Débora Kulczar - Site Institucional

Este projeto é uma landing page institucional para o escritório Débora Kulczar Advocacia.

## ✅ Estrutura do Projeto
```
Debora_Site/
├── index.html
├── style.css
└── assets/
    ├── logo.png
    ├── escritorio.jpg
    └── profile.jpg
```

### **Arquivos**
- `index.html` → Estrutura completa do site.
- `style.css` → Estilos com cores, layout e responsividade.
- Pasta `assets` → Contém as imagens do site:
  - **logo.png** → Logotipo.
  - **escritorio.jpg** → Imagem do escritório.
  - **profile.jpg** → Foto profissional.

---

## ✅ Passo a Passo para Publicar no GitHub Pages

### 1. **Criar Repositório no GitHub**
- Acesse [GitHub](https://github.com) e crie um novo repositório.
- Nome sugerido: `Debora_Site`.

### 2. **Enviar os Arquivos**
- Extraia o pacote ZIP.
- Confirme que a estrutura está igual ao exemplo acima.
- Faça upload de todos os arquivos (`index.html`, `style.css` e pasta `assets`) para o repositório.

### 3. **Ativar GitHub Pages**
- Vá em **Settings** → **Pages**.
- Em **Branch**, selecione `main` e a pasta raiz (`/`).
- Clique em **Save**.

### 4. **Acessar o Site**
- Após alguns minutos, o site estará disponível em:
```
https://<seu-usuario>.github.io/Debora_Site/
```

### 5. **Forçar Atualização**
- Caso não veja as mudanças, limpe o cache do navegador ou pressione **Ctrl + F5**.

---

## ✅ Dicas Importantes
- **Nomes das imagens devem ser exatos**: `logo.png`, `escritorio.jpg`, `profile.jpg`.
- **Links no HTML** já estão configurados para usar a pasta `assets`.
- Para alterar cores ou fontes, edite o arquivo `style.css`.
