# Landing Page Responsiva

Este projeto contém uma landing page completa com HTML, CSS e JavaScript, seguindo os requisitos:

✅ Logo no canto esquerdo do banner
✅ Botões estilizados em azul
✅ Banner full screen com sobreposição azul
✅ Layout responsivo e elegante

---

## Estrutura dos Arquivos
- **index.html**: Estrutura principal da página.
- **style.css**: Estilos visuais e responsividade.
- **script.js**: Funções JavaScript para interatividade.

---

## Como Usar
1. **Baixe e extraia** o arquivo `landing_page.zip`.
2. Coloque os arquivos em uma pasta do seu projeto.
3. Adicione sua imagem de banner no arquivo `style.css`:
   ```css
   background: url('banner.jpg') no-repeat center center/cover;
   ```
   Substitua `banner.jpg` pela imagem desejada.
4. Substitua `logo.png` no HTML pela sua logo.

---

## Personalização
- **Cores**: Ajuste no arquivo `style.css` (ex.: `.btn-primary` para botões).
- **Textos**: Edite diretamente no `index.html`.
- **Funções JS**: No `script.js`, personalize as ações dos botões.

---

## Como Executar Localmente
1. Abra a pasta do projeto.
2. Clique duas vezes no arquivo `index.html` ou abra com um servidor local (ex.: Live Server no VS Code).

---

## Requisitos
- Navegador atualizado.
- Editor de código (opcional, recomendado VS Code).
