# Site Débora Kulczar Advocacia

## Como usar
1. Exporte as imagens do Figma (logo, background, ícones) e coloque na pasta `assets/`.
2. Abra `index.html` no navegador para visualizar.

## Publicar no GitHub Pages
1. Crie um repositório no GitHub.
2. Faça upload de todos os arquivos.
3. Vá em **Settings → Pages**.
4. Selecione:
   - Branch: `main`
   - Folder: `/root`
5. Salve e aguarde alguns minutos.

## Estrutura
- `index.html`: Página principal
- `style.css`: Estilos com responsividade
- `assets/`: Pasta para imagens
